5.4.2.3. statsmodels.sandbox.formula.Quantitative¶

class statsmodels.sandbox.formula.Quantitative(name, func=None, termname=None, transform=<function <lambda>>)[source]¶

A subclass of term that can be used to apply point transformations of another term, i.e. to take powers:

>>> import numpy as np
>>> from nipy.fixes.scipy.stats.models import formula
>>> X = np.linspace(0,10,101)
>>> x = formula.Term('X')
>>> x.namespace={'X':X}
>>> x2 = x**2
>>> print np.allclose(x()**2, x2())
True
>>> x3 = formula.Quantitative('x2', func=x, transform=lambda x: x**2)
>>> x3.namespace = x.namespace
>>> print np.allclose(x()**2, x3())
True
