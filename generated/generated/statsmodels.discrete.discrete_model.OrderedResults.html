3.2.2.3.30. statsmodels.discrete.discrete_model.OrderedResults¶

class statsmodels.discrete.discrete_model.OrderedResults(model, mlefit, cov_type='nonrobust', cov_kwds=None, use_t=None)[source]¶

A results class for ordered discrete data.

Parameters:

model : A DiscreteModel instance

params : array-like

The parameters of a fitted model.

hessian : array-like

The hessian of the fitted model.

scale : float

A scale parameter for the covariance matrix.
Returns:

Attributes

aic : float

Akaike information criterion. -2*(llf - p) where p is the number of regressors including the intercept.

bic : float

Bayesian information criterion. -2*llf + ln(nobs)*p where p is the number of regressors including the intercept.

bse : array

The standard errors of the coefficients.

df_resid : float

See model definition.

df_model : float

See model definition.

fitted_values : array

Linear predictor XB.

llf : float

Value of the loglikelihood

llnull : float

Value of the constant-only loglikelihood

llr : float

Likelihood ratio chi-squared statistic; -2*(llnull - llf)

llr_pvalue : float

The chi-squared probability of getting a log-likelihood ratio statistic greater than llr. llr has a chi-squared distribution with degrees of freedom df_model.

prsquared : float

McFadden’s pseudo-R-squared. 1 - (llf / llnull)
