6.5.5.3.3. statsmodels.sandbox.nonparametric.kernel_extras.LeaveOneOut¶

class statsmodels.sandbox.nonparametric.kernel_extras.LeaveOneOut(X)[source]¶

Generator to give leave-one-out views on X.

Parameters:

X : array-like

2-D array.

Notes

A little lighter weight than sklearn LOO. We don’t need test index. Also passes views on X, not the index.

Examples

>>> X = np.random.normal(0, 1, [10,2])
>>> loo = LeaveOneOut(X)
>>> for x in loo:
...    print x
