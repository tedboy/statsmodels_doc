3.9.3.1.2. statsmodels.regression.mixed_linear_model.MixedLMParams¶

class statsmodels.regression.mixed_linear_model.MixedLMParams(k_fe, k_re, use_sqrt=True)[source]¶

This class represents a parameter state for a mixed linear model.

Parameters:

k_fe : integer

The number of covariates with fixed effects.

k_re : integer

The number of covariates with random effects.

use_sqrt : boolean

If True, the covariance matrix is stored using as the lower triangle of its Cholesky square root, otherwise it is stored as the lower triangle of the covariance matrix.

Notes

This object represents the parameter state for the model in which the scale parameter has been profiled out.
