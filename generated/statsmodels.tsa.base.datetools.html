4.8.12.1. Functions¶

date_range_str(start[, end, length]) Returns a list of abbreviated date strings.
dates_from_range(start[, end, length]) Turns a sequence of date strings and returns a list of datetime.
dates_from_str(dates) Turns a sequence of date strings and returns a list of datetime.
