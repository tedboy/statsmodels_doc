4.2.2.1.6. statsmodels.datasets.utils.webuse¶

statsmodels.datasets.utils.webuse(data, baseurl='http://www.stata-press.com/data/r11/', as_df=True)[source]¶
Parameters:

data : str

Name of dataset to fetch.

baseurl : str

The base URL to the stata datasets.

as_df : bool

If True, returns a pandas.DataFrame
Returns:

dta : Record Array

A record array containing the Stata dataset.

Notes

Make sure baseurl has trailing forward slash. Doesn’t do any error checking in response URLs.

Examples

>>> dta = webuse('auto')
